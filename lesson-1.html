Git source control

Git helps you to secure your code against loss and unwanted changes.

You can go back to any saved state and you can easily collaborate with your team.

Check that Git is installed by:

  1. Open Terminal
  2. Navigate to your directory: cd ~/REDIBFD/my-first-javascript
  3. Initialize a Git repository: git init
