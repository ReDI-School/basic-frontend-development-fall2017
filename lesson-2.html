Loops

Loops are handy, if you want to run the same code over and over again!

Instead of:

text += cars[0] + "<br>"; text += cars[1] + "<br>"; text += cars[2] + "<br>"; text += cars[3] + "<br>"; text += cars[4] + "<br>"; text += cars[5] + "<br>";

We can write:

for (i = 0; i < cars.length; i++) { text += cars[i] + "<br>"; }

Types of loops

for
for/in
while
do/while

For Loops

for (let i = 0; i < [number]; i++) { [...] }

Example:

for (let i = 0; i < [number]; i++) { [...] }

Trying for loops

Change your script.js to the following:

window.onload = function () { let name = window.prompt('Whats your name?'); let number = window.prompt('Set a number (not too big!)'); let message = ''; for(let i = 0; i < number; i++) { message += name + '<br>'; } document.body.innerHTML = message; };
